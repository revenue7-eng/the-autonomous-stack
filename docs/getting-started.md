# Getting Started with TAS

This guide helps you quickly understand how to use The Autonomous Stack.

---

## 1. Understand the Philosophy

Start with [Philosophy](philosophy.md) to grasp the core concepts:  
- **Pause**, **Exit**, **Recoverability** – the three ethical principles.  
- **Autonomy Level** (A0–A3) and **Transparency Level** (T0–T2) – how we evaluate technologies.  
- **whose.world** – the philosophical foundation.

If you're short on time, read the one‑page [Assessment Scale](catalog/assessment-scale.md) for definitions.

---

## 2. Browse the Catalog

The [Technology Catalog](catalog/) lists technologies by architectural layer:

- Network, Identity, Storage, Observability, Compute, Security, Applications

Each card shows:
- **Autonomy Level** (how independent from cloud)
- **Transparency Level** (how open/auditable)
- **Philosophical Assessment** (specific criteria)

Use the catalog to pick tools that match your autonomy requirements.

---

## 3. Build a System

The [Recipes](recipes/) provide step‑by‑step instructions for building complete autonomous stacks.

Start with the [Minimal Autonomous Server](recipes/minimal-server.md) – it combines several core technologies into a working system.

---
