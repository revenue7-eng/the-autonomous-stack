# Technology Catalog

This catalog evaluates technologies according to two independent scales:

- **Autonomy Level** (A0–A3): operational independence from cloud and internet.
- **Transparency Level** (T0–T2): architectural openness (documented or open source).

Each card also includes the **Philosophical Assessment** based on *whose.world* criteria: Pause, Exit, Recoverability, Visibility, External Dependencies.
