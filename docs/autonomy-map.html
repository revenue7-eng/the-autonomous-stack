A living star chart of the infrastructure technology landscape. Every evaluated tool is plotted as a star — its brightness and size reflect autonomy, its color shows trajectory. Stars cluster into 12 constellations defined by the combination of Autonomy and Transparency level.

The 12 Constellations

Lux A3/T2

Full autonomy, open source. The brightest stars — you own everything, code included. The TAS ideal.

Vigil A3/T1

Fully autonomous, documented. Operates offline, code not fully open but architecture is transparent.

Solitudo A3/T0

Autonomous but opaque. Your data, your control — but you cannot see inside the tool.

Libertas A2/T2

Nearly free, open source. One dependency remains but the code is fully auditable.

Stella A2/T1

Mostly autonomous, documented. A reliable intermediate — conscious trade-off zone.

Janua A2/T0

Data is yours, tool is closed. A half-open door — autonomy without transparency.

Aurora A1/T2

Dependent but open source. Cloud-reliant yet auditable — heading toward freedom.

Iter A1/T1

Partial freedom, partial light. A path — not a destination.

Nebula A1/T0

Partially dependent, opaque. Blurred edges — hard to know what you control.

Paradoxa A0/T2

Cloud-bound yet open source. Rare and contradictory — dependent infrastructure with visible code.

Vinculum A0/T1

Dependent but documented. The chains are visible. You know what holds you.

Umbra A0/T0

Full dependency, full opacity. The darkest corner — no control, no visibility.

The map is open and growing. Today 27 stars, 12 constellations. As new tools are evaluated, new stars appear. As new criteria emerge, new constellations form. The map is a separate project — it uses TAS data but lives beyond it.
